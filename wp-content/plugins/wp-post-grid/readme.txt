Efficient way to show post images in a grid of your choice. You can select from a range of grid design to show your posts.

== Description ==

WP Post Grid is used for showing your post's image in a grid. You can grab image from any particular posts and show it through the grid. WP Post Grid offers 8 different types of grid options. Comfortable to use, easy to navigate and very much simple to build.

Please check the Demo for real understanding.


= Features =

1. Show posts in different styles
2. 8 different types of grid options to choose from.
3. You can select most recent post, published post or any custom image with link.
4. Image can be linked with a short header/line.
5. Fully responsive.
6. Use shortcode to show it anywhere.

== Installation ==

Using The WordPress Dashboard

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for WP Post Grid'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

Uploading in WordPress Dashboard

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select wp-post-grid.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

Using FTP

1. Download wp-post-grid.zip
2. Extract the wp-post-grid directory to your computer
3. Upload the waving-portfolio directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard

== Screenshots ==

1. 8 different types of grid options.

2. Select "Default (Most Recent Post)" to show recent post and save options.

3. Select "Published Post", search your required post and save options.

4. Select "Custom Image". Add image url, link, short headline and save options. Make sure to keep the aspect ratio of the image is 2:1, i.e. width should be twice of the height.

5. For every grid there are options to select default or published post or custom images. For example, for 4 grids, there are four posts options.

6. Add shortcode to any post/page to active the plugin.
